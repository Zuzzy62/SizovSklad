Приложение для автоматизации складских процессов ИП "Сизов А. Н."
--- О приложении ---
Мобильное приложение для автоматизации складских процессов, приёмка, отгрузка, инвентаризация. Приложение поддерживает четыре роли пользователей: младший кладовщик, старший кладовщик, менеджер, руководитель.
Для всех пользователей:
•	Вход в приложение по логину и паролю;
•	Интуитивно понятный интерфейс;
Для младшего кладовщика:
•	Осуществлять приёмку товаров;
•	Осуществлять отгрузку товаров;
Для старшего кладовщика:
•	Редактирование приёмки товара;
•	Подтверждение операций;
Для руководителя:
•	Просмотр списка всех сотрудников;
•	Редактирование данных любого сотрудника;
--- Технические особенности ---
Язык программирования: Java
Минимальная версия Android: API 21 (Android 5.0)
Используемые библиотеки:
AndroidX AppCompat и CardView
--- Установка и запуск ---
Клонируйте репозиторий
Откройте проект в Android Studio
Соберите проект (Build -> Make Project)
Запустите на эмуляторе или реальном устройстве
